## About True Crime Metrix

True Crime Metrix is an advanced CRUD application focussed on the collation of news articles, available media resources and data relating to high-profile crimes and court cases.

Infomation collected on True Crime Metrix

- Crime categories
- Criminal cases
- News articles
- Criminals
- Judges
- Lawyers
- Victims
- Witnesses

True Crime Metrix is currently accessible to a closed group of private users.
