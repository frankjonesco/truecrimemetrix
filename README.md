# True Crime Metrix
#### Welcome to the Github repository for the True Crime Metrix project.

## About True Crime Metrix

True Crime Metrix is records keeping project concerned with the documenting of relevant information pertaining to some of the highest profile criminal cases and televised trials in the world.

The focus is in the building of a extensive knowledge bank regarding the perpatrators, victims, attorneys, Judges, as well as the circumstances, surrounding culture and speculative theories.
